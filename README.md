### Custom-JSON-Parser
A proof of concept custom JSON parser for my portfolio website

##The Logic:
Based on the railroad diagrams JSON Syntax. Also known as McKeeman Form:

```
json
  element

value
  object
  array
  string
  number
  "true"
  "false"
  "null"

object
  '{' whitespace '}'
  '{' members '}'

```
